# 💰 FinanceIQ

Sistema de análisis financiero inteligente desarrollado con Django.

## 🚀 Descripción

FinanceIQ es una aplicación web que permite registrar ingresos y gastos diarios, analizar el comportamiento financiero del usuario y generar recomendaciones personalizadas, proyecciones futuras y alertas inteligentes.

El sistema utiliza un motor de análisis propio basado en lógica en Python, sin depender completamente de APIs externas de IA.

---

## 🧠 Características principales

* 📊 Dashboard financiero en tiempo real
* 🧾 Registro de ingresos y gastos
* 🏷️ Clasificación automática de gastos
* 📉 Análisis de salud financiera
* 💡 Recomendaciones inteligentes
* 🔮 Proyecciones a futuro (3, 6 y 12 meses)
* 🎯 Gestión de objetivos financieros
* 🚨 Detección de anomalías y alertas

---

## 🛠️ Tecnologías utilizadas

* Python 3.11
* Django
* Django REST Framework
* PostgreSQL
* Tailwind CSS

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/Lfandino98/financeiq.git
cd financeiq
```

2. Crear entorno virtual:

```bash
python -m venv venv
venv\Scripts\activate
```

3. Instalar dependencias:

```bash
python -m pip install -r requirements.txt
```

4. Ejecutar migraciones:

```bash
python manage.py migrate
```

5. Iniciar servidor:

```bash
python manage.py runserver
```
